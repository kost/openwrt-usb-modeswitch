# openwrt-usb-modeswitch

Put into feeds.conf
```
src-git usbmodeswitch git://github.com/kost/openwrt-usb-modeswitch.git
```

So, you will have something like this:
```
src-git packages git://git.openwrt.org/12.09/packages.git
src-git usbmodeswitch git://github.com/kost/openwrt-usb-modeswitch.git
```
